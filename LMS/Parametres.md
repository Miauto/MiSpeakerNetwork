# LMS
## Pre-requis
- Logitech Media Server
- Plugins
  - UPnP/DLNA bridge ([Philippe](https://github.com/philippe44/LMS-to-uPnP))
  - [Materiel skin](https://github.com/CDrummond/lms-material) (optionel)
  - Spotify

## Configuration UPnP


| Parametre         | Valeur      |
| ----------------- | ----------- |
| HTTP mode         | No length   |
| Seek on resume    | No          |
| Codecs            | flc,aif |
| Send ICY metadata | No          |
|                   |
